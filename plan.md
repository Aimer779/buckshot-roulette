# 恶魔轮盘网页版 - 执行计划

## 目标
调研恶魔轮盘(Buckshot Roulette)游戏玩法，开发并部署一个可玩的网页版。

## Stage 1: 调研
- 加载 `deep-research-swarm` 技能
- 部署多个研究agent调研：
  - 游戏基本规则和流程
  - 道具系统
  - 回合机制
  - 胜负条件
  - 多人/AI对战模式
- 输出：详细的游戏设计参考文档

## Stage 2: 设计与准备
- 加载 `vibecoding-webapp-swarm` 技能
- 基于调研结果设计网页版游戏：
  - 简化但核心的游戏机制
  - 视觉风格（暗黑、紧张、俄罗斯轮盘氛围）
  - 玩家 vs AI 的对战模式
- 输出：游戏设计文档 + 技术方案

## Stage 3: 开发
- 使用 React + TypeScript + Tailwind 构建游戏
- 核心功能：
  - 霰弹枪装填机制（实弹/空包弹）
  - 射击选择（对自己/对对手）
  - 道具系统（放大镜、手铐、香烟、饮料等）
  - 生命值系统
  - AI对手
  - 回合管理
  - 音效和视觉反馈
- 输出：可运行的网页游戏代码

## Stage 4: 部署
- 构建生产版本
- 部署到线上环境
- 输出：可访问的URL

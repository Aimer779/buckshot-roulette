# 🎰 Buckshot Roulette 恶魔轮盘（网页版）

一个基于 React + TypeScript + Vite 构建的《恶魔轮盘》（Buckshot Roulette）网页版游戏。

玩家将与 AI 庄家在 1998 年的地下夜总会里进行一场生死对决：实弹与空包弹被随机装入霰弹枪，你需要利用道具、推理与运气，在三轮对决中活下来。

🔗 **在线体验**：https://aimer779.github.io/buckshot-roulette/ *(如已部署)*  
📦 **GitHub 仓库**：https://github.com/Aimer779/buckshot-roulette

---

## ✨ 游戏特性

- **经典三轮对决**：教学局 → 道具局 → 决胜局，难度与策略逐步升级
- **实弹 / 空包弹机制**：每轮随机装填，明确显示数量但顺序未知
- **9 种核心道具**：放大镜、手铐、香烟、啤酒、手锯、肾上腺素、过期药品、逆变器、一次性手机
- **智能 AI 对手**：根据桌面可见信息决策，会使用道具，公平对战
- **暗黑视觉风格**：地下夜总会氛围，像素风 UI 与金属质感
- **音效反馈**：射击、上膛、道具使用等关键动作带有音效

## 🎮 核心玩法

| 动作 | 实弹 (Live) | 空包弹 (Blank) |
|------|-------------|----------------|
| 对自己射击 | 自己扣 1 血，回合交换 | 无事发生，**额外一回合** |
| 对对手射击 | 对手扣血，回合交换 | 无事发生，回合交换 |

### 回合配置

| 回合 | 生命 | 每轮道具数 | 特殊机制 |
|------|------|-----------|----------|
| 第一回合 | 2 格 | 0 | 无道具教学 |
| 第二回合 | 4 格 | 2 | 引入道具系统 |
| 第三回合 | 5+ 格 | 4 | 闸刀机制：低血量时无法回血 |

### 道具说明

- **放大镜** 🔍：查看当前枪膛内子弹类型
- **手铐** ⛓️：跳过对手下一回合
- **香烟** 🚬：恢复 1 点生命
- **啤酒** 🍺：弹出当前子弹（不发射）
- **手锯** 🪚：下一发实弹伤害翻倍（2 点）
- **肾上腺素** 💉：偷取对手一个道具并立即使用
- **过期药品** 💊：50% 恢复 2 血 / 50% 扣 1 血
- **逆变器** 🔄：切换当前子弹类型（实弹 ↔ 空包弹）
- **一次性手机** 📱：告知枪内某随机位置子弹类型

## 🛠 技术栈

- **框架**：React 19 + TypeScript
- **构建工具**：Vite 7
- **样式**：Tailwind CSS 3 + shadcn/ui 组件库
- **状态管理**：Zustand
- **路由**：React Router 7
- **动画**：Framer Motion / GSAP
- **音效**：Howler.js
- **包管理器**：pnpm

## 🚀 本地运行

> 需要 Node.js 20+ 和 pnpm

```bash
# 进入项目目录
cd buckshot-roulette/app

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

开发服务器默认运行在 http://localhost:5173/

### 构建生产版本

```bash
pnpm build
```

构建产物输出在 `app/dist/` 目录，可直接部署到任意静态托管服务。

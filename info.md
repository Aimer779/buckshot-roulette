# 恶魔轮盘 (Buckshot Roulette) 游戏调研报告

## 游戏概述
《恶魔轮盘》是由 Mike Klubnika 开发、CRITICAL REFLEX 发行的桌面恐怖策略游戏。核心概念是将俄罗斯轮盘赌用12号口径泵动式霰弹枪进行暗黑改造。2023年12月首次发布，2024年4月登陆Steam。

背景设定：1998年地下夜总会，玩家与神秘"庄家"进行生死对决。

## 核心玩法

### 三回合制对决
- 第一回合（教学）：双方2格生命，无道具。装弹：1实2空 → 3实2空
- 第二回合（道具）：双方4格生命，每轮2个道具。装弹：1实1空 → 逐步增加
- 第三回合（决胜）：双方5格生命（实为4+2隐藏），每轮4个道具，引入闸刀机制

### 霰弹枪机制
- 每轮开始时装入实弹（红色）和空包弹（黑色）
- 明确告知实弹和空包弹数量，但顺序完全随机
- 最多容纳8发子弹
- 枪内子弹打完后重新装填

### 射击选择（核心博弈）
- **对自己射击 + 空包弹** = 无事发生，获得额外回合（高收益）
- **对自己射击 + 实弹** = 自己扣1血，回合交换（高风险）
- **对对手射击 + 实弹** = 对手扣血，回合交换
- **对对手射击 + 空包弹** = 无事发生，回合交换

### 闸刀机制（第三回合）
- 当任意一方电量降至2格或以下，触发闸刀
- 除颤仪电线被切断，对应区域黑屏
- 无法使用任何道具恢复电量
- 再受一次实弹攻击即直接死亡

## 道具系统（9种核心道具）

### 基础道具（5种）
1. **放大镜** (T0/S级) - 查看当前子弹类型
2. **手铐** (T0/S级) - 跳过对手下一回合
3. **香烟** (T0/S级) - 恢复1点生命
4. **啤酒** (T1/B级) - 弹出当前子弹（不发射）
5. **手锯** (T0/S级) - 下一发实弹伤害翻倍(2点)

### 加倍模式专属（4种）
6. **肾上腺素** (T0/S级) - 偷取对手一个道具并立即使用
7. **过期药品** (T2/C级) - 50%恢复2血 / 50%扣1血
8. **逆变器** (T0.5/A级) - 切换当前子弹类型(实弹↔空弹)
9. **一次性手机** (T1/B级) - 告知枪内某随机位置子弹类型

### 道具规则
- 每回合内道具使用数量不限
- 最多持有8个道具
- 道具使用后立即消失（一次性）

## AI对手行为

### 三种行为状态
- **进攻状态**：实弹>空包弹时，AI必定对你射击
- **防御状态**：空包弹>实弹时，AI对自己射击以排出空包弹
- **不确定状态**：实弹=空包弹时，AI进行50/50随机决策

### AI公平性
- AI不使用隐藏信息，只根据桌面可见信息决策
- 和人类玩家拥有完全相同的信息集
- AI也会使用道具

## 视觉风格
- 复古像素风，类似《Inscryption》
- 桌面第一人称视角
- 暗色调：黑色、深灰色为主，实弹的红色最醒目
- 地下夜总会设定：金属栏杆、昏暗灯光
- 极简UI设计，"世界内"界面系统
- 工业电子乐配乐

## 音效设计
- 霰弹枪泵动声、子弹上膛声、扳机扣动声
- 空包弹的咔嗒声
- 心跳声、金属碰撞声
- 工业电子乐OST

## 经典道具Combo
1. 放大镜+手锯 = 确认实弹后双倍伤害（2点斩杀）
2. 放大镜+手铐+手锯 = 控制+双倍伤害+额外射击
3. 放大镜+逆变器+手锯 = 空弹转实弹+双倍伤害
4. 手铐+肾上腺素 = 控制+掠夺

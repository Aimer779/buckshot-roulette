# 恶魔轮盘 (Buckshot Roulette) 道具系统深度研究报告

> 研究时间：2025年  
> 游戏版本：V1.2+ (Steam版)  
> 游戏模式覆盖：故事模式 / 加倍模式 (Double or Nothing) / 多人模式 (Multiplayer)

---

## 一、道具系统概述

《恶魔轮盘》(Buckshot Roulette) 是一款由独立开发者 Mike Klubnika 制作的恐怖策略游戏。道具系统是游戏的核心策略要素，将纯粹的运气博弈转化为深度的资源管理与决策对抗。

### 道具系统核心规则

| 规则项 | 说明 |
|--------|------|
| **道具获取** | 每轮子弹重新装填后，根据存活玩家数量随机补给道具 [^30^] |
| **道具上限** | 玩家最多同时拥有 **8个** 道具 [^12^] [^59^] |
| **使用限制** | 每回合内道具使用数量不限，一个道具只能使用一次 [^69^] |
| **第一局** | 无道具，纯概率博弈 [^66^] |
| **第二局** | 每轮获取 **2个** 道具 [^66^] |
| **第三局** | 每轮获取 **4个** 道具 [^66^] |
| **道具消耗** | 使用后立即消失（一次性消耗品） |

### 道具分类体系

游戏将道具分为三大类别 [^28^]：

- **信息类道具 (Information Items)**：减少不确定性，将猜测转化为确定性决策
- **节奏类道具 (Tempo Items)**：改变行动顺序，控制谁可以下一步行动
- **伤害类道具 (Damage Items)**：改变伤害输出，使正确的选择更具价值

## 二、基础模式道具详解

### 1. 放大镜 (Magnifying Glass)

#### 基本信息

| 属性 | 详情 |
|------|------|
| **中文名称** | 放大镜 |
| **英文名称** | Magnifying Glass |
| **可用模式** | 故事模式 / 加倍模式 / 多人模式 |
| **道具类型** | 信息类 |
| **玩家评级** | **T0 / S级** |

#### 具体效果

使用后可以**查看当前枪膛内装填的子弹类型**（实弹或空包弹），被查看的子弹不会被移除或消耗 [^3^] [^9^]。放大镜提供的是关于"当前待发射子弹"的即时信息。

#### 战略使用建议

- **最优先使用时机**：回合开始时，在其他任何道具使用之前先用放大镜确认子弹类型 [^25^]
- **核心决策依据**：看到实弹 → 射向对手；看到空弹 → 射向自己（获得额外回合）[^53^]
- **配合手锯**：先用放大镜确认实弹，再使用手锯，可确保造成双倍伤害 [^25^] [^53^]
- **配合啤酒**：先用放大镜确认是空弹，再用啤酒将其弹出，避免浪费可能的好子弹 [^53^]

#### 局限性与注意事项

放大镜并非万能救命道具。在以下情况中放大镜无效：

> 当你只剩1格血量，而枪里还剩两发实弹时——手铐能救你、香烟能救你、逆转器能救你、甚至啤酒也能救你，但是放大镜就没办法救你。[^10^] [^16^]

#### 与其他道具的配合

| 配合道具 | 配合效果 |
|---------|---------|
| 手锯 | 确认实弹后使用手锯，确保双倍伤害命中对手 |
| 手铐 | 确认实弹后先用手铐铐住对手，再安全射击 |
| 啤酒 | 确认空弹后用啤酒弹出，为后续子弹创造更好局面 |
| 逆变器 | 确认空弹后用逆变器转为实弹，配合手锯打出2点伤害 [^18^] |

### 2. 手铐 (Handcuffs)

#### 基本信息

| 属性 | 详情 |
|------|------|
| **中文名称** | 手铐 |
| **英文名称** | Handcuffs |
| **可用模式** | 故事模式 / 加倍模式 |
| **道具类型** | 节奏类（控制类） |
| **玩家评级** | **T0 / S级** |

#### 具体效果

使用后可以**跳过对手（庄家/恶魔）的下一个回合**，使对方在该回合内完全无法行动（不能开枪、不能使用道具、不能跳过）[^3^] [^11^] [^66^]。

手铐效果**可以叠加**——如果双方都有手铐，理论上可以连续使用 [^50^]。

**使用限制**：不能对已经被铐住的对手再次使用 [^19^]。

#### 战略使用建议

- **最优使用时机**：当你确认或高度确信下一发是实弹时，使用手铐可以确保自己连续两回合都能攻击对手 [^15^]
- **保命用途**：在血量危急时使用，给自己一个喘息的机会 [^25^]
- **压制用途**：当你和对手都有强力道具时，先手使用手铐可以打断对手的道具连锁
- **概率控制**：当实弹数量多于空弹时，用手铐后连续射击两次，提高命中实弹的概率 [^53^]

#### 庄家AI使用模式

庄家（Dealer）将手铐视为 **Tier 3（控制级）** 道具优先级。AI行为特征：
- 如果枪内剩余2发或以上子弹，会立即使用 [^50^]
- 从不在最后一发子弹时使用 [^50^]
- 不会"保留"手铐等待更好时机——有就用 [^50^]

#### 与其他道具的配合

| 配合道具 | 配合效果 |
|---------|---------|
| 放大镜 | 先用放大镜确认实弹，再用手铐，确保连续两发都能命中对手 [^15^] |
| 手锯 | 先用放大镜+手锯组合，再用手铐，实现"双倍伤害+额外回合"的秒杀组合 |
| 啤酒 | 当局面不明朗时，先用手铐获得喘息机会，再用啤酒排除危险子弹 |

### 3. 香烟 (Cigarette Pack)

#### 基本信息

| 属性 | 详情 |
|------|------|
| **中文名称** | 香烟/卷烟/烟 |
| **英文名称** | Cigarette Pack / Cigarettes |
| **可用模式** | 故事模式 / 加倍模式 / 多人模式 |
| **道具类型** | 恢复类 |
| **玩家评级** | **T0 / S级** |

#### 具体效果

使用后**恢复1点生命值**（1格除颤仪电量）[^3^] [^11^]。

**重要限制**：
- 香烟**不能超出回合开始时的最大生命值**。例如，如果回合开始时有4格血，使用香烟后最多回到4格，不能到达5格 [^53^]
- 在第三局中，如果触发了"假血"机制（电量条最边缘的两格），香烟在该局剩余时间内**无法回复血量** [^3^] [^11^]

#### 战略使用建议

- **及时使用**：受到损伤后应尽快使用，不要留到"紧急情况"——因为对手可能用肾上腺素偷走你的香烟 [^12^]
- **生命值管理**：保持3点及以上生命值的玩家胜率显著高于持续低血量作战者 [^71^]
- **避免过度囤积**：过度堆叠香烟会挤占情报类或控制类道具的携带空间 [^71^]
- **第三局特别注意**：注意假血机制，一旦触发假血，香烟即失效 [^3^]

#### 与其他道具的配合

| 配合道具 | 配合效果 |
|---------|---------|
| 过期药品 | 先用香烟恢复1点血量作为"保险"，再使用过期药品——即使药品失败扣1血，仍有香烟的恢复做缓冲 [^53^] |
| 肾上腺素 | 当对手有香烟时，用肾上腺素偷取并立即使用，同时阻止对手恢复 |

### 4. 啤酒/饮料 (Beer/Drink)

#### 基本信息

| 属性 | 详情 |
|------|------|
| **中文名称** | 啤酒/饮料/饮品 |
| **英文名称** | Beer / Drink / Bloxy Cola |
| **可用模式** | 故事模式 / 加倍模式 / 多人模式 |
| **道具类型** | 节奏类（退弹类） |
| **玩家评级** | **T1 / A级** |

#### 具体效果

使用后**将当前枪膛内的子弹安全退出**（不发射该子弹），可以看到退出的子弹类型 [^4^] [^9^]。

使用啤酒**不会结束你的回合**——你可以在使用啤酒后继续行动 [^31^]。

**特殊情况**：如果使用啤酒时枪内只剩最后一发子弹，该回合结束 [^4^]。

#### 战略使用建议

- **最优使用窗口**：枪内剩余 **3-4发** 子弹时使用效果最佳。此时可以有效控制局面，将不可控变为可控 [^10^] [^77^]
- **信息获取**：当枪内只剩2发子弹时，使用啤酒相当于一个"变体的放大镜"——排除一发后，你就知道另一发的类型了 [^4^]
- **紧急避险**：当局面高度不利时，用啤酒移除已知的实弹
- **连续使用**：一回合内可以连续使用多个啤酒，直到枪内没有子弹或啤酒用完 [^31^]
- **概率重构**：使用后改变了剩余子弹的比例结构，可能创造更有利的概率

#### 局限性与注意事项

> 一般只会在子弹还剩三到四发的时候，让啤酒来把局面控制到你可控的地步，其余的时候当你需要喝下啤酒的时候，说明你基本只剩下啤酒了。[^10^]

在子弹总量较多（如初始6发）阶段，优先选择手铐、放大镜等高功能性道具更为合理 [^77^]。

#### 与其他道具的配合

| 配合道具 | 配合效果 |
|---------|---------|
| 放大镜 | 先用放大镜确认是空弹，再用啤酒弹出，避免浪费 |
| 手铐 | 当局面混乱时，先用手铐控制节奏，再用啤酒排除危险子弹 |
| 手锯 | 当实弹比例较高时，先用啤酒精简枪膛，再用放大镜+手锯打出精准伤害 |

### 5. 手锯 (Hand Saw / Saw)

#### 基本信息

| 属性 | 详情 |
|------|------|
| **中文名称** | 手锯/锯子/折叠刀 |
| **英文名称** | Hand Saw / Saw / Handsaw |
| **可用模式** | 故事模式 / 加倍模式 / 多人模式 |
| **道具类型** | 伤害类 |
| **玩家评级** | **T0 / S级** |

#### 具体效果

使用后锯断霰弹枪枪管，使**下一发实弹的伤害从1点提升至2点**（双倍伤害）[^3^] [^9^] [^70^]。

**重要说明**：
- 手锯效果**仅对实弹生效**——如果下一发是空弹，手锯效果不触发也不消耗 [^69^]
- 手锯效果**不可叠加**——连续使用两个手锯不会使伤害变为3点
- 无论实弹还是空弹，射击一次后霰弹枪会自动恢复原状 [^76^]
- 每次使用消耗1个手锯

#### 战略使用建议

- **最优使用时机**：在**已经确认下一发是实弹**的情况下使用（通过放大镜、推理或已知信息）[^25^]
- **快速击杀**：当对手只有2点血量时，手锯+实弹可一击必杀 [^45^]
- **避免浪费**：不要在不知道子弹类型的情况下盲目使用手锯——如果下一个是空弹，手锯效果就会被浪费
- **双刃剑**：手锯同样适用于对手——如果对手使用了手锯，你也可以被双倍伤害击中

#### 与其他道具的配合

| 配合道具 | 配合效果 |
|---------|---------|
| 放大镜 | **最经典配合**：先用放大镜确认实弹，再用手锯，确保2点伤害命中 [^25^] [^53^] |
| 手铐 | 先用手铐控制对手，再用放大镜+手锯组合，实现"双倍伤害+额外回合" |
| 肾上腺素 | 偷取对手的手锯，在确认实弹后使用，打出意想不到的2点伤害 [^33^] |
| 逆变器 | 先用放大镜确认空弹，用逆变器转为实弹，再用手锯，打出2点伤害 [^18^] |

## 三、加倍模式 (Double or Nothing) 专属道具详解

### 6. 肾上腺素 (Adrenaline)

#### 基本信息

| 属性 | 详情 |
|------|------|
| **中文名称** | 肾上腺素 |
| **英文名称** | Adrenaline |
| **可用模式** | 加倍模式 / 多人模式 |
| **道具类型** | 节奏类（偷取类） |
| **玩家评级** | **T0 / S级** |

#### 具体效果

使用后可以**从对手的道具栏中偷取一个道具，并立即使用该道具** [^9^] [^19^]。

**重要规则限制**：
- 使用肾上腺素后只有 **10秒** 选择要偷取的道具 [^19^]
- **不能偷取另一个肾上腺素**（不可偷取同款道具）[^19^]
- 如果对手已经被铐住，**不能偷取手铐** [^19^]
- 如果在10秒内未选择道具，效果消失，无法偷取 [^19^]
- 偷取的道具会**立即自动使用**

#### 战略使用建议

- **优先偷取目标**：
  - **S级选择**：手铐（控制节奏）、放大镜（获取信息）、手锯（双倍伤害输出）[^8^] [^53^]
  - **A级选择**：香烟（恢复生命，同时阻止对手恢复）
  - **B级选择**：啤酒（排除危险子弹，同时精简对手资源）
  - **避免选择**：过期药品（高风险，偷过来可能反而害自己）

- **心理博弈**：使用肾上腺素不仅是获得一个道具，更是**阻止对手使用该道具** [^25^]
- **提前规划**：由于只有10秒选择时间，**在使用肾上腺素之前就要想好要偷什么** [^19^]
- **反制策略**：当你有肾上腺素时，应考虑先用掉自己的高价值道具（如手锯、香烟），避免被对手偷走 [^19^]

#### 庄家AI的肾上腺素行为

有趣的是，庄家使用肾上腺素时，动画表现为"直接把肾上腺素扔掉"而非注射——这暗示了庄家只是一个头+手的存在，没有身体可以注射 [^19^] [^24^]。

#### 与其他道具的配合

| 配合道具 | 配合效果 |
|---------|---------|
| 手铐 | 先用手铐控制对手，再用肾上腺素偷取对手道具，对手既无法行动又失去资源 |
| 过期药品 | 当对手有过期药品时，用肾上腺素偷取并迫使其使用——如果失败，对手扣血 |
| 多个肾上腺素 | 一回合内可使用多个肾上腺素偷取多个道具 [^26^] |

### 7. 过期药品 (Expired Medicine)

#### 基本信息

| 属性 | 详情 |
|------|------|
| **中文名称** | 过期药品/过期药物/神秘小药丸 |
| **英文名称** | Expired Medicine / Expired Gummies / Pills |
| **可用模式** | 加倍模式 / 多人模式 |
| **道具类型** | 恢复类（风险类） |
| **玩家评级** | **T2 / C级** |

#### 具体效果

使用后进行一次50/50的判定 [^9^] [^20^]：
- **50%概率**：恢复 **2点** 生命值
- **50%概率**：失去 **1点** 生命值

**历史版本差异**：在V1.2.1更新前，过期药品的概率为40%恢复/60%扣血 [^20^]。

#### 战略使用建议

- **最后手段**：仅在走投无路、没有其他恢复手段时使用 [^20^]
- **安全血量使用**：建议在有2点或以上血量时使用——即使失败扣1血也不会立即死亡 [^53^]
- **绝不在1血时使用**：仅剩1点血量时使用过期药品，失败即意味着死亡 [^53^]
- **"让对手偷"策略**：有玩家认为最好的用法是放在道具栏让对手用肾上腺素偷走——有较大概率让对手消耗一个肾上腺素并扣血 [^10^]
- **连续使用策略**：如果有多包过期药品，连续使用可以提高至少成功一次的概率（连续3次全部失败的概率仅6.4%）[^70^]

#### 庄家与过期药品

如果庄家使用过期药品失败，会表现为"倒下"并失去1点血量 [^20^] [^24^]。

#### 与其他道具的配合

| 配合道具 | 配合效果 |
|---------|---------|
| 香烟 | 先用香烟恢复1点血量作为"保险垫"，再使用过期药品——即使失败也有缓冲 [^53^] |
| 肾上腺素 | 被对手偷走时可能害对手扣血，但也可能让对手恢复2血——双刃剑 |

### 8. 逆变器/逆转器 (Inverter)

#### 基本信息

| 属性 | 详情 |
|------|------|
| **中文名称** | 逆变器/逆转器/反转器 |
| **英文名称** | Inverter / Polarity Switch |
| **可用模式** | 加倍模式 / 多人模式 |
| **道具类型** | 信息/改变类 |
| **玩家评级** | **T0.5 / A级** |

#### 具体效果

使用后**切换当前枪膛内子弹的类型**：
- 实弹 → 变为空包弹
- 空包弹 → 变为实弹 [^9^] [^18^]

**注意**：逆变器只改变**当前待发子弹**的类型，不会改变所有子弹 [^31^]。

#### 战略使用建议

- **保命策略**：当你知道下一发是实弹且会击中你时，用逆变器将其转为空弹 [^10^]
- **斩杀策略**：先用放大镜确认是空弹，用逆变器转为实弹，配合手锯打出2点伤害 [^18^] [^25^]
- **拖回合策略**：将实弹转为空弹后射向自己，获得额外回合，继续获取道具 [^18^]
- **终局处理**：当最后一发是空弹时，用逆变器转为实弹，射向对手结束回合 [^33^]

#### 注意事项

> 要小心枪内剩下空弹时，庄家也会转换成实弹来打你。[^10^]

逆变器是双向的——你也可以 inadvertently 给对手创造机会。

#### 与其他道具的配合

| 配合道具 | 配合效果 |
|---------|---------|
| 放大镜 | **最经典斩杀组合**：先用放大镜确认空弹→逆变器转为实弹→手锯双倍伤害→射向对手=2点伤害 [^18^] |
| 手锯 | 确认实弹后用逆变器转空弹打自己（额外回合），或确认空弹后转实弹+手锯打对手 |
| 肾上腺素 | 偷取对手的逆变器，配合自己的放大镜和手锯实现斩杀 |

### 9. 一次性手机 (Burner Phone)

#### 基本信息

| 属性 | 详情 |
|------|------|
| **中文名称** | 一次性手机/燃烧手机/电话 |
| **英文名称** | Burner Phone / Mobile Phone |
| **可用模式** | 加倍模式 / 多人模式 |
| **道具类型** | 信息类 |
| **玩家评级** | **T1 / B级** |

#### 具体效果

使用后，一个神秘的声音会告诉你**枪内某个随机位置子弹的类型**（实弹或空包弹），格式为"第X发子弹是实弹/空包弹" [^9^] [^22^]。

**重要规则**：
- 不会告诉你**当前待发子弹**的信息（从第二发开始计数）[^22^]
- 在单人模式中，枪内必须**剩余2发或以上**子弹才能使用；如果只剩1发，手机会返回"How Unfortunate..."（真不幸...）并浪费该道具 [^22^]
- 在多人模式中，枪内必须**剩余3发或以上**才能使用 [^22^]
- **一次性手机不受逆变器影响**——逆变器不会改变手机已经获取的信息 [^53^]
- 多次使用可以获取多个位置的信息

#### 战略使用建议

- **早期使用**：建议在回合早期、子弹数量较多时使用——这样获取的信息可以在后续多个决策中使用 [^25^]
- **辅助推理**：配合已发射的子弹进行推理，推算出当前待发子弹的类型
- **不使用在终局**：当子弹所剩无几时，不如直接用放大镜
- **不连续多次使用**：一回合内多次使用手机只能获取相同信息 [^34^]
- **心理威慑**：即使手机没有给你有用的信息，对手不知道这一点——可能误认为你已经掌握了全局 [^31^]

#### 与其他道具的配合

| 配合道具 | 配合效果 |
|---------|---------|
| 放大镜 | 手机告诉你未来的某发子弹，放大镜告诉你当前的子弹——形成完整信息链 |
| 啤酒 | 用手机排除已知的危险子弹，用啤酒精确移除 |
| 手铐 | 掌握信息后用手铐控制节奏，确保信息转化为实际伤害 |

## 四、多人模式 (Multiplayer) 独占道具详解

### 10. 干扰器 (Jammer)

#### 基本信息

| 属性 | 详情 |
|------|------|
| **中文名称** | 干扰器/干扰机 |
| **英文名称** | Jammer |
| **可用模式** | 多人模式（独占） |
| **道具类型** | 节奏类（控制类） |
| **道具定位** | 多人模式版"手铐" |

#### 具体效果

使用后**选择一个对手，使其跳过下一个回合** [^38^] [^40^]。

干扰器是多人模式中对**手铐的替代道具**——因为多人模式下需要指定跳过哪个玩家的回合 [^38^]。

**限制**：
- 不能选择自己作为目标 [^40^]
- 不能选择已死亡的玩家作为目标 [^40^]

#### 战略使用建议

- **针对领先者**：跳过当前最强的玩家的回合，给自己创造优势 [^31^]
- **打断道具链**：当某个玩家即将使用强力道具组合时，用干扰器打断
- **自保用途**：当你知道下一发子弹危险且即将轮到某个玩家行动时，用干扰器跳过其回合
- **与手铐策略相同**：干扰器的基本策略逻辑与手铐一致 [^40^]

### 11. 遥控器 (Remote)

#### 基本信息

| 属性 | 详情 |
|------|------|
| **中文名称** | 遥控器 |
| **英文名称** | Remote |
| **可用模式** | 多人模式（独占） |
| **道具类型** | 节奏类（顺序类） |

#### 具体效果

使用后**反转当前的回合顺序**（顺时针变为逆时针，或反之）[^33^] [^38^]。

**重要说明**：在只有2名玩家的对局中，遥控器**没有任何效果**——因为反转顺序和不反转是一样的 [^33^]。

#### 战略使用建议

- **多人局核心道具**：在3-4人对局中，遥控器可以彻底改变局势走向
- **保护盟友**：如果你有盟友或想保护某个玩家，用遥控器改变顺序让危险回合转到对手身上
- **反制高价值目标**：当某个即将行动的玩家持有强力道具（如手锯+放大镜组合）时，用遥控器改变顺序
- **配合干扰器**：先用遥控器改变顺序，再用干扰器控制关键目标

> 评估每个玩家持有的道具，然后利用遥控器打乱对手的计划，让局势转为对你有利。[^33^]

## 五、道具组合与高级策略

### 经典道具Combo（连招）

#### Combo 1：放大镜 + 手锯（信息斩杀）

**使用顺序**：放大镜 → 确认实弹 → 手锯 → 射向对手
**效果**：造成 **2点伤害**，一击可能结束回合
**适用场景**：对手血量≤2时，或需要快速削减对手血量
**评级**：⭐⭐⭐⭐⭐（最稳定的高伤Combo）

#### Combo 2：放大镜 + 手铐 + 手锯（控制斩杀）

**使用顺序**：放大镜 → 确认实弹 → 手铐 → 手锯 → 射向对手 → 再次射击
**效果**：造成 **2点伤害 + 额外1点伤害**，共3点伤害
**适用场景**：对手血量≤3时
**评级**：⭐⭐⭐⭐⭐（最强斩杀Combo）

#### Combo 3：放大镜 + 逆变器 + 手锯（变废为宝）

**使用顺序**：放大镜 → 确认空弹 → 逆变器 → 转为实弹 → 手锯 → 射向对手
**效果**：将原本无用的空弹变为2点伤害的实弹
**适用场景**：枪内空弹较多、实弹较少时
**评级**：⭐⭐⭐⭐（需要加倍模式道具）

#### Combo 4：手铐 + 肾上腺素（控制+掠夺）

**使用顺序**：手铐 → 控制对手 → 肾上腺素 → 偷取对手道具 → 使用
**效果**：对手既无法行动又失去一个道具
**评级**：⭐⭐⭐⭐（强力压制Combo）

#### Combo 5：过期药品 + 香烟（风险缓冲）

**使用顺序**：香烟 → 恢复1血 → 过期药品 → 50%恢复2血 / 50%扣1血
**效果**：即使药品失败，仍有香烟的恢复做缓冲
**评级**：⭐⭐⭐（安全使用过期药品的方法）

#### Combo 6：啤酒 + 放大镜 + 手锯（精简斩杀）

**使用顺序**：啤酒 → 移除不确定子弹 → 放大镜 → 确认实弹 → 手锯 → 射向对手
**效果**：在精简后的枪膛中确保双倍伤害命中
**评级**：⭐⭐⭐⭐

### 全道具使用成就 (Full House)

在加倍模式中，有一个成就要求**在一回合内使用全部9种道具** [^26^] [^5^]：

- 需要在自己的道具栏中拥有8种不同道具（包括肾上腺素）
- 使用肾上腺素偷取对手的第9种道具
- 在一回合内将全部9种道具全部使用
- 这需要极好的运气和精心的规划

## 七、各模式道具策略要点

### 故事模式（3局制）

| 阶段 | 血量 | 道具数 | 策略重点 |
|------|------|--------|---------|
| 第一局 | 2血 | 无 | 纯概率博弈，积累对规则的理解 |
| 第二局 | 4血 | 每轮2个 | 学习道具配合，优先使用放大镜和手铐 |
| 第三局 | 5血(实为4+假血) | 每轮4个 | 注意假血机制，注意香烟在假血触发后失效 |

### 加倍模式（无尽模式）

- 道具种类更多（9种 vs 基础5种），策略深度更大
- 肾上腺素改变了游戏的核心meta——道具管理成为胜负关键
- 过期药品的引入增加了高风险高回报的博弈元素
- 一次性手机提供了跨回合的信息优势
- 需要管理8个道具栏位，优先级判断更重要

### 多人模式

- 从1v1变为最多4人混战，社交博弈和心理战成为核心
- 干扰器替代手铐，可以精确选择跳过哪个对手
- 遥控器可以改变回合顺序，在3-4人局中策略价值极高
- 需要关注多个对手的道具状态
- 人类对手比AI更难预测，道具的欺骗性和心理威慑更大

## 八、核心策略总结

### 道具使用的黄金法则

1. **信息优先**：一回合开始时，优先使用信息类道具（放大镜、手机），再执行行动 [^12^]
2. **确认后输出**：永远不要在手锯之前不确认子弹类型——浪费手锯是最致命的错误
3. **及时恢复**：受伤后立即使用香烟恢复，不要等到紧急情况——对手可能偷走它 [^12^]
4. **控制节奏**：手铐不是只有在你知道是实弹时才用——在局面混乱时用它获得喘息机会
5. **概率思维**：持续追踪已发射的子弹数量，用概率指导决策
6. **反被偷意识**：当你有高价值道具时，考虑先用掉或保护好——对手可能有肾上腺素 [^19^]
7. **风险管控**：过期药品只在安全血量时使用，绝不在1血时使用
8. **道具清空**：回合结束前尽量用完有价值的道具——未使用的道具在回合结束后会被清空

### 常见致命错误

- 在不确定子弹类型时使用手锯 [^8^]
- 把香烟留到"紧急时刻"使用，结果被对手偷走 [^12^]
- 使用肾上腺素偷取低价值道具（如啤酒）而非高价值道具（如手铐、放大镜）[^8^]
- 在枪内只剩1-2发时使用啤酒而非信息道具 [^10^]
- 在1血时使用过期药品 [^53^]
- 忘记追踪子弹数量，把每个决策都当作50/50来对待 [^8^]
